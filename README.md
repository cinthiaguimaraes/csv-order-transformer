# CSV Order Transformer

Projeto de estudo em Python para ler um arquivo CSV de pedidos, aplicar uma transformação simples e gerar um novo arquivo para análise.

## Objetivo

Aprender fundamentos de Python, manipulação de arquivos CSV, organização de projetos e Git.
